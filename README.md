# 📊 Customer Churn Prediction Dashboard

## 📌 Project Overview
This project focuses on analyzing customer behavior and identifying patterns that lead to customer churn using Power BI.

The goal is to help businesses:
- Identify high-risk customers
- Improve retention strategies
- Make data-driven decisions


## 🛠️ Tools & Technologies
- Power BI
- Data Analysis
- Data Visualization
